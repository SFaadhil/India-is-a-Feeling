TESTIMONIAL / EXPERIENCE VIDEOS
================================
Drop the final video files into THIS folder (videos/) using EXACTLY these names:

  gv-testimonial.mp4    ->  Global Volunteer page  (global-volunteer.html)
  gta-testimonial.mp4   ->  Global Talent page     (global-talent.html)
  gte-testimonial.mp4   ->  Global Teacher page    (global-teacher.html)

Until a file is added, the page shows a styled "Testimonial video coming soon"
placeholder automatically. As soon as the correctly-named file exists, the
video player appears in its place — no code changes needed.

Recommended format: MP4 (H.264 video + AAC audio), 1920x1080 (16:9 landscape).

OPTIONAL poster frames (thumbnail shown before the video is played):

  gv-poster.jpg
  gta-poster.jpg
  gte-poster.jpg

(1920x1080 JPG. If omitted, the video's first frame is used.)
